Core workflow

From hero idea to saved result in two API calls.

The product flow is simple on purpose: create a hero, run a search mode, then reopen the saved run later from history.

01 Save a hero from any ticker and date range.
02 Run Price, Economic, or Social DNA against that hero.
03 Reopen the exact ranked run from saved history.
1const hero = await api.post("/api/heroes", {

6
7const run = await api.post(
